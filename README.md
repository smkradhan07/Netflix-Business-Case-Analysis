# Netflix Business Case Analysis

## Objective
Analyze Netflix content trends and provide strategic business recommendations.

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib

## Project Structure
- Q1: Problem Statement & Basic Metrics
- Q2: Data Cleaning & Overview
- Q3: Non-Graphical Analysis
- Q4: Visual Analysis
- Q5: Missing Value & Outlier Check
- Q6: Analytical Insights
- Q7: Business Insights
- Q8: Final Recommendations

## Key Insights
- Majority content post 2015
- TV Shows drive retention
- Mature audience focus increasing
- Strong regional specialization
